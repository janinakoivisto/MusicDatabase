# MusicDatabase

## Johdanto

Projektin aihe on järjestelmä omille lempibiiseille.

## Järjestelmän määrittely

MusicDatabase-sovellukseen kuuluu kaksi erilaista käyttäjäryhmää/roolia. 

## Roolit

Järjestelmästä löytyy kaksi eri roolia, normaali käyttäjä ja ylläpitäjä. Vain ylläpitäjän on mahdollista poistaa kappaleita. Normaali käyttäjä pystyy lisäämään ja muokkaamaan kappaleita, tarkastelemaan eri genrejä ja lisäämään niitä. Ylläpitäjältä löytyy nämä samat oikeudet.

## Sovelluksen toiminta

Sovellus toimii niin, että käyttäjä joko luo uudet tunnukset tai kirjautuu olemassaolevilla tunnuksilla sisään. Tämän jälkeen käyttäjä ohjautuu kappalesivulle, josta löytyy biisit listattuna. Käyttäjä voi sitten joko lisätä tai muokata kappaleita, tai siirtyä tarkastelemaan eri genrejä.
